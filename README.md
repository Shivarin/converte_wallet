# converte_wallet

> **Статус: в разработке** — основной API и логика кошелька активно пишутся.

Лёгкая Python-библиотека для конвертации валют без привязки к конкретному провайдеру курсов. Разработана так, чтобы встраиваться в любой проект — веб-бэкенд, CLI-скрипт, Telegram-бот, торговый скрипт — с минимальной настройкой.

---

## Зачем это нужно

Большинство конвертеров валют либо:
- жёстко завязаны на один API-провайдер и ломаются при смене тарифов/лимитов
- возвращают просто число без контекста (какой курс? когда получен?)
- смешивают логику конвертации с HTTP-запросами, что делает тестирование болью

`converte_wallet` разделяет ответственность: **получение курсов** — это подключаемый провайдер, **конвертация** — чистая математика, **кошелёк** — опциональный слой состояния поверх.

## Как будет выглядеть использование

Это целевой интерфейс — показывает, к чему движется разработка.

```python
from converte_wallet import Converter
from converte_wallet.providers import ExchangeRateAPIProvider

provider = ExchangeRateAPIProvider(api_key="ВАШ_КЛЮЧ")
converter = Converter(provider=provider)

result = converter.convert(100, "USD", "EUR")
print(result.amount)      # 92.31
print(result.rate)        # 0.9231
print(result.fetched_at)  # 2026-04-09T14:00:00Z
print(result.provider)    # "exchangerate-api"
```

Пакетная конвертация:
```python
results = converter.convert_many([
    (100, "USD", "EUR"),
    (50,  "GBP", "JPY"),
    (1,   "BTC", "USD"),
])
```

Кошелёк:
```python
from converte_wallet import Wallet

wallet = Wallet(converter=converter, base_currency="USD")
wallet.deposit(500, "EUR")
wallet.deposit(0.01, "BTC")

print(wallet.total_value())              # итого в USD
wallet.convert_balance("EUR", "GBP", amount=100)
```

Статический провайдер (без сети, удобно для тестов):
```python
from converte_wallet.providers import StaticProvider

provider = StaticProvider(rates={"USD": {"EUR": 0.92, "GBP": 0.79}})
converter = Converter(provider=provider)
```

## Куда встраивается

Библиотека рассчитана на интеграцию в:

- **Любой Python веб-фреймворк** — FastAPI, Django, Flask: один раз создать `Converter` при старте, пробросить куда нужно
- **Telegram / Discord боты** — провайдер поддерживает async
- **CLI-скрипты** — нулевая настройка через переменные окружения
- **Jupyter ноутбуки** — объекты результатов совместимы с pandas
- **Микросервисы** — опциональный REST-сервер, готов к Docker
- **Офлайн / изолированные среды** — `StaticProvider` не требует сети

## Конфигурация

Все чувствительные данные читаются из переменных окружения — никаких конфиг-файлов для базового использования:

```env
CONVERTE_PROVIDER=exchangerate-api
CONVERTE_API_KEY=ваш_ключ
CONVERTE_CACHE_TTL=3600       # секунды, по умолчанию 3600
CONVERTE_BASE_CURRENCY=USD
```
